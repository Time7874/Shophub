## 📖 About The Project

ShopHub is a responsive multi-page e-commerce website built using **HTML, CSS, and Vanilla JavaScript**.

It includes:

- Dynamic product rendering
- Shopping cart functionality
- User settings with dark mode
- LocalStorage data persistence
- Responsive UI design

This project demonstrates frontend development fundamentals and clean UI implementation.

---

## ✨ Features

- 🛒 Add to Cart System
- 💾 LocalStorage Cart Persistence
- 🌙 Dark Mode Settings
- ⚙️ User Preference Saving
- 📱 Fully Responsive Layout
- 🧭 Multi-page Navigation
- 🎨 Modern UI Design
